Welcome to Basic RPG!

Basic RPG is an RPG style grinding game where you beat monsters to become stronger!

Do you get bored from grinding? Don't worry, this game makes it even more boring by having no graphics! But at least there's some different mechanics to make up for it I guess :P

  • Kill monsters and level up to become stronger!
  • Get gold and become rich!
  • Die a horrible death I guess?

There is now a dedicated Discord Server for the game!

This game is in development, after an update, your save data may be wiped, YOU HAVE BEEN WARNED

You can play the modern version here!

Don't want to play with the modern combat system, you masochistic/lucky bastard? The legacy version of the game is here, but it will no longer be updated.

Combat Overhaul Extension Update

Because the combat overhaul just wasn't enough for me.

  • New Feature: Traits
    • You can tier up traits up to 3 times using Trait Crystals, found in the shop for 100 each, getting more expensive as you buy them.
      • Vigor grants 5/7/10% HP when killing monsters
      • Agility grants a 3/5/10% chance to dodge attacks
      • Unrelenting grants a 1/3/5% chance to not get knocked unconcious when you normally would

MAJOR UPDATE - Combat Overhaul

Remember that old combat system where it was completely random? Not anymore!

  • Dexterity now has an effect

  • No more vulnerability hits
    • You can finally heal in peace!
  • Reworked monster attack damage
  • Optimized the combat a LOT
  • You lose less gold from your stash when fleeing before rolling initiative, or (see Lively Adventures Update) when you win initiative
