Hello! This is where my Twine games live! You can find their source HTML files on the GitHub Repository!
So far I only have one: Basic RPG, which is an unfinished RPG style grinding game, with the following mechanics:
  • Stats to improve your character
  • A semi-complete Inventory System
  • An unfinished shop to buy Health Potions
  • A traits system
  • Encouragement to adventure longer with better rewards
  • Monsters become more dangerous the longer you go
  • EXP system to level up
  • EXP requirement increases as you level up
  • A time system (although not that in depth)
  • An achievements system
  • Saving and Loading
  • Something special might happen if your HP reaches zero during an adventure?

Note: I might not ever finish this game.
